切换页面

操作具体是W键选1，S键选2，空格确认。

但主要方法是：SceneManager.LoadScene(1);

通过这个方法成功把小组作业的两个游戏拼在一起了，很满足。
需要注意的是我们需要先倒入LoadScene，具体方法是：
File -> BildSettings –> 依次按顺序托入场景 –> 叉掉
